# Elite Cuts - Barbar Shop 💈

Hey there! 👋 This is a modern, responsive website I built for a local barbershop here in Nairobi called "Elite Cuts." I used it to master semantic HTML5, clean CSS flexbox/grid layouts, and basic scroll animations.

## 📸 Preview
![Barbar Shop Preview](screenshots/preview.png)

## Why I built this
A lot of small businesses here don't have websites and just rely on Instagram. I wanted to build a landing page that looks highly professional and premium, matches the masculine vibe of a high-end barber studio (with warm copper and wood accents), and includes a clear, easy-to-use services catalog. I hand-coded all the CSS and animations from scratch to keep the code lightweight and learn the foundations properly!

## Features
- Fully responsive services catalog showing pricing (Classic Haircut, Beard Trim, Combo)
- Elegant gallery grid showcasing styles and the shop interior
- Working hours listing and contact form section
- Smooth fade-in scroll animations using a custom JavaScript `IntersectionObserver`
- Styled dark theme with warm orange accents

## Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Grid & Flexbox)
- **Interactivity:** Vanilla JavaScript
