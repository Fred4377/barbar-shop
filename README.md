# 💈 Elite Cuts Barbershop - Retro Cream & Espresso Webpage

![Preview](screenshots/preview.png)

A vintage retro-styled promotional website for a local barbershop, designed with a **Retro Cream, Espresso & Gold** color scheme.

## ✨ Humanized & Localized Features
- 🇰🇪 **KSh Price List** — Local haircut services priced realistically in Kenyan Shillings (e.g. Fade at KSh 300, Beard Trim at KSh 150).
- 🕒 **Barber Shift availability Board** — Real-time board showing active barbers (Fred, Mike, John) and their active shift status.
- 📍 **Nairobi Address Localization** — Address and location links set to Nairobi CBD.

## 🛠️ Tech Stack
- **Frontend:** HTML5, Vanilla CSS, JavaScript
- **Typography:** Playfair Display & Merriweather
